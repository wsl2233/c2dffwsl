# C²DFF-Net 使用指南

## 📁 项目文件说明

| 文件 | 说明 |
|------|------|
| `C2DFF.yaml` | C²DFF-Net 网络配置文件 |
| `yolov8-twoCSP-6-4.yaml` | Baseline 模型配置 |
| `block.py` | 核心模块实现（CDFIM、CGSA等） |
| `ALM.py` | 自适应光照感知掩码数据增强 |
| `C2DFF_Drone.pt` | DroneVehicle 数据集预训练权重 |
| `C2DFF_VEDAI.pt` | VEDAI 数据集预训练权重 |
| `C2DFF_FLIR.pt` | FLIR 数据集预训练权重 |

## 🚀 快速开始

### 1. 环境要求

已确认环境配置：
- ✅ Python 3.8.10
- ✅ PyTorch 2.0.0+cu118
- ✅ Ultralytics 8.4.37
- ✅ CUDA 11.8 + RTX 3090

### 2. 替换 Ultralytics 的 block.py

由于项目使用了自定义模块，需要将 `block.py` 替换到 Ultralytics 库中：

```bash
# 找到 ultralytics 安装路径
python -c "import ultralytics; print(ultralytics.__file__)"

# 备份原文件
cp /path/to/ultralytics/nn/modules/block.py /path/to/ultralytics/nn/modules/block.py.bak

# 替换为项目的 block.py
cp /root/C2DFF-Net/block.py /path/to/ultralytics/nn/modules/block.py
```

**注意：** 如果加载预训练权重时遇到模块名称错误，根据 README 说明：
- 将 `CDFIM` 替换为 `CPCA_CF`
- 将 `CGSA` 替换为 `PolarizedSelfAttention_Channel`

### 3. 使用预训练模型进行推理

```python
from ultralytics import YOLO

# 加载预训练模型
model = YOLO('C2DFF_Drone.pt')

# 进行推理
results = model('path/to/image.jpg')

# 显示结果
for r in results:
    r.show()
```

### 4. 训练自己的模型

```python
from ultralytics import YOLO

# 加载模型配置
model = YOLO('C2DFF.yaml')

# 开始训练
results = model.train(
    data='your_data.yaml',
    epochs=100,
    imgsz=640,
    batch=16,
    device='0'
)
```

## 📊 数据集准备
